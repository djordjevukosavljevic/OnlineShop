Online Shop Application
Overview
This project is an online shop application built using ASP.NET Core 8.0 MVC. The application includes functionalities like product browsing, shopping cart management, and order processing. Currently, user registration forms are being developed.

Features
Product Listing and Details: View available products and see detailed information.
Shopping Cart Management: Add, update, or remove products from your cart.
Order Processing: Complete your purchase and manage orders.
Admin Panel for Product Management: Admin users can manage product listings, prices, and inventory.
