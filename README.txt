Online Shop Application<br/>
- - - - - - - - - - - - - - -
Overview<br/>
This project is an online shop application built using ASP.NET Core 8.0 MVC. It features product browsing, shopping cart functionality, and order processing.<br/>
Currently doing on users registrations forms.<br/>
- - - - - - - - - - - - - - -
Features<br/>
Product Listing and Details:<br/>

Shopping Cart Management<br/>
Order Processing<br/>
Admin Panel for Product Management<br/>
